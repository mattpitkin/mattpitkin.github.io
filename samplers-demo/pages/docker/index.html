Docker images

Fri 09 February 2018

I have created Docker images, available on DockerHub, which have all the samplers included in this demonstration installed.

The images with the codes installed using a Python 2, Python 3, or a Jupyter notebook server (containing both Python 2 and 3 kernels) can be downloaded using:

docker pull mattpitkin/samplers:python2
docker pull mattpitkin/samplers:python3
docker pull mattpitkin/samplers:jupyter

The mattpitkin/samplers:python2 can be used to run, e.g., the emcee test script, assuming the script is downloaded to ${HOME}, with:

docker run -it -v ${HOME}:/work mattpitkin/samplers:python2

A Jupyter server to run, e.g., the notebook from which the demonstration page was created, can be run using:

docker run -it -v {notebook_location}:/home/jovyan/work -e NB_UID=1001 --user root --rm -p 8888:8888 mattpitkin/samplers:jupyter

where {notebook_location} is the local path under which the notebooks you want to run live.
